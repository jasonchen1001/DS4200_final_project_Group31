Project Introduction

This project analyzes electric vehicle (EV) charging stations across the United States to understand their distribution, accessibility, and characteristics. Our goal is to provide insights into the current state of EV charging infrastructure and identify areas for improvement.

About the Data

The dataset contains information about EV charging stations with the following key attributes:

  • Location (Latitude, Longitude)
  • Charger Type (L1, L2, DC)
  • Availability Hours
  • Cost per kWh
  • User Ratings
  • Installation Year
